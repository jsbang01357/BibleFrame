# 가톨릭 성경 데이터 검토

## 2026-07-13 조사 결과

- 현재 BibleFrame 본문은 eBible.org의 `Korean Bible 1910`이며 가톨릭 정경이 아닌 66권이다.
- 한국천주교주교회의의 2005년 공용 번역본 `성경`은 가톨릭식 73권과 확장된 에스테르기·다니엘서를 제공한다.
- 주교회의는 본문 사용량과 관계없이 어문·음향·영상·컴퓨터 데이터 이용에 사전 승인을 요구한다.
- 같은 안내에서 개인 홈페이지나 인터넷 카페에는 저작권 사용을 승인하지 않는다고 명시한다.
- 따라서 `bible.cbck.or.kr` 본문을 수집해 GitHub, RAG, 검색 데이터 또는 TTS 음원으로 재배포하지 않는다.
- 조사한 공개 데이터 중 한국어 가톨릭 73권 전체를 명시적인 자유 이용 조건으로 제공하는 판본은 확인하지 못했다.
- eBible.org의 `World English Bible (Catholic)`은 가톨릭식 73권과 순서를 갖춘 영어 Public Domain 판본이다.
- 2026-05-22 VPL ZIP은 창세기 1,533절이 빠진 72권이므로 사용하지 않는다.
- 같은 배포본의 USFM ZIP은 창세기를 포함한 73권, 1,328장, 35,408개 절 레코드로 검증했다.
- WEB-C의 요엘 3장·말라키 4장 체계는 본문을 바꾸지 않고 가톨릭식 요엘 4장·말라키 3장 장절로 변환한다.
- USFM ZIP SHA-256: `b2807cd86cd2423214e5c51bc1b92217ceaf9420cfade84ff951766cd3e2aa4b`

## 확인한 공식 자료

- 한국천주교주교회의 성경: <https://bible.cbck.or.kr/knb>
- 「성경」 저작권 사용 안내: <https://cbck.or.kr/Board/K7300/13000715>
- World English Bible (Catholic): <https://ebible.org/bible/details.php?id=eng-web-c>

## 가능한 경로

1. **권장: 공개 원문 기반 가톨릭 길잡이**
   - `World English Bible (Catholic)` 73권을 검색·RAG 근거로 사용한다.
   - 화면에는 가톨릭식 한국어 책 이름과 장·절을 표시한다.
   - 실제 한국어 본문 읽기는 해당 장의 CBCK 공식 페이지로 이동시킨다.
   - CBCK 번역문, 임베딩, TTS 음원은 저장하거나 재배포하지 않는다.
2. **정식 한국어 본문 사용**
   - 한국천주교중앙협의회에 별도 이용 주체와 서비스 형태로 승인을 신청한다.
   - 개인 홈페이지는 승인하지 않는다는 현행 안내 때문에 현재 형태로는 가능성이 낮다.
3. **자체 공개 번역 제작**
   - Public Domain 가톨릭 원문을 한국어로 새로 번역하고 신학·번역 감수를 거친다.
   - 교회 공인 번역이 아니므로 검수 전에는 `가톨릭 성경`이나 `공인 번역`으로 표시하지 않는다.

## 선택한 경로

- Public Domain WEB-C USFM 73권을 Qwen으로 한국어 새 번역한다.
- `Qwen3-14B-FP8` L4 시험은 품질과 처리속도가 부족해 모델을 언디플로이했다.
- GCP Vertex AI의 관리형 `Qwen3-Next-80B-A3B-Instruct` MaaS로 번역하고 원문 대조 2차 교정을 수행한다.
- 절 ID 보존, 한국어 출력, 누락, 원문 혼입을 자동 검사하고 책별 체크포인트로 중단 후 재개한다.
- 개별 실패 절은 별도 큐에 격리해 전체 실행을 보호한 뒤 마지막에 재처리한다.
- 결과는 `BibleFrame 73권 공개 번역 초안`과 `가톨릭 정경 기반 · 비공인 기계 번역 초안`으로 표시한다.
