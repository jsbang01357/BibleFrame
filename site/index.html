HELP & INFORMATION

자주 묻는 질문

BibleFrame의 본문, 검색, RAG 자료에 관한 안내입니다.

어떤 성경 번역을 사용하나요?

eBible.org가 Public Domain으로 제공하는 Korean Bible 1910을 사용합니다. 현대 번역과 표현이 다를 수 있습니다.

검색어가 서버에 저장되나요?

아니요. 검색과 성경 읽기는 내려받은 데이터로 사용자의 브라우저 안에서 처리됩니다.

RAG ZIP은 어디에 쓰나요?

장 단위 JSONL 본문을 ChatGPT 프로젝트나 벡터 데이터베이스 등에 넣어 개인용 성경 질의 시스템을 만들 때 사용할 수 있습니다.

글자를 더 크게 볼 수 있나요?

성경 브라우저 상단의 ‘가−’, 퍼센트, ‘가＋’ 버튼으로 본문 크기를 조절할 수 있습니다. 선택한 크기는 이 기기에 기억됩니다.

본문에 빠진 부분이 있나요?

제공 원본은 베드로전서가 4장 14절에서 끝납니다. 다른 번역을 임의로 섞지 않고 원본 상태를 그대로 밝힙니다.

본문 안내

이 사이트는 eBible.org가 Public Domain으로 제공하는 1910 한국어 성경을 사용합니다. 자세한 검수 기록은 GitHub 저장소에서 확인할 수 있습니다.
